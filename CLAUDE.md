# Project Brief

## Goal
Build a premium website that sells shirt and merchandise design packs.

## Product Packs
- Starter Pack: 10 designs
- Growth Pack: 50 designs
- Ultimate Pack: 101 designs

## Audience
- print-on-demand sellers
- merch creators
- small clothing brands
- Etsy / Shopify sellers
- creators who need ready-to-use graphics

## Design Direction
- premium dark UI
- modern 2026 aesthetic
- black background
- subtle cyan accent
- sharp typography
- clean spacing
- strong hierarchy
- minimal but visually strong
- not playful
- not generic
- not cheap-looking

## UX Goals
- users must understand the offer within 3 seconds
- users must clearly see what each pack contains
- the difference between 10, 50, and 101 packs must be obvious
- preview sections must reduce uncertainty
- the website must feel trustworthy and premium
- the layout must support conversion

## Tech Requirements
- React
- Vite
- Tailwind CSS
- reusable components
- responsive design
- clean code
- fast loading
- minimal dependencies

## Content Rules
- all sales copy in English
- no lorem ipsum
- no filler text
- no fake testimonials
- no overcomplicated animations

## Sections Required
1. Hero
2. Pack cards
3. Preview section
4. What's included
5. Comparison table
6. FAQ
7. Final CTA
